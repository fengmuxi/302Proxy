Overview

配置概览

路径前缀模式

每个路径前缀都在自己的规则区里维护地区匹配开关。开关关闭时，直接走该前缀的默认规则。

在线定位策略

在线 IP 定位源支持多源配置、按权重轮询，主源异常时只回退一次，再落到离线 IP 库。

规则组织方式

同一路径前缀下可以配置多条目标地址规则，其中至少建议保留一条默认规则作为兜底。

建议操作顺序

先新增路径前缀，再新增该前缀下的转发规则，最后按需要开启该前缀的地区匹配。
